Allows you to manage your excerpt settings through a WordPress admin page.

== Description ==

This plugin gives you an admin management page under "Settings" which will allow you to manage all of your excerpt settings in one place, without the need for modifying your theme files. It will override any theme or plugin excerpt settings, by simply overriding them. You can manage everything: Excerpt length, Read More text, pre-Read More text, post-read more text, and whether or not you show read more text.

== Installation ==

1. Upload sk_excerpt directory to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Go to Settings > Custom Excerpts to modify your excerpt settings.

== Frequently Asked Questions ==

= What is this plugin for? =

To allow you to dynamically change your excerpt settings for your website.

= Will this override my theme's excerpt settings? =

Yes, absolutely. You can use this with any theme and you will have full control over the settings for your excerpts.
